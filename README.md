# El Morjene Market - Mobile Version

Version mobile optimisée du site e-commerce El Morjene.

## Optimisations mobiles

- **Menu burger** pour navigation simplifiée
- **Layout une colonne** adapté aux petits écrans
- **Touches agrandies** pour faciliter la saisie
- **Navigation tactile** avec gestes fluides
- **Images optimisées** pour mobile
- **Dark mode** supporté

## Structure

- `index.html` - Page principale
- `css/styles.css` - Styles responsive mobiles
- `js/app.js` - Logique d'application mobile
- `uploads/` - Images des produits

## Lancer

```bash
npx http-server
# Puis ouvre http://localhost:8080
```

Ou double-clique simplement sur `index.html`.

## Déploiement

Deploy sur Netlify comme la version desktop.
